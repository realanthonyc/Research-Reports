1. **AI infrastructure 敘事從 GPU/HBM 擴散到 TPU、CPU、Arm host CPU 與電力架構**

   - 訊號：Google 發表 TPU 8t/8i，訓練與推理開始更明確分工；NVIDIA 與 Google Cloud 擴大 Rubin-powered A5X、Blackwell confidential computing、Gemini on Distributed Cloud 等合作；Counterpoint 指出 Arm-based CPUs 在 custom AI ASIC server host CPU 市場份額可能由約 2025 年 25% 升至 2029 年 90%。社群同時反覆討論 agentic AI 讓 CPU processing 從 chatbot 時代的配角變成 research agent / orchestration 的瓶頸。
   - 解讀 / 疑點：這是今日最強的結構性訊號。市場不只在買 NVDA，而是在重新分配 AI capex 受益者：GOOGL/AVGO/MRVL 是 ASIC/TPU 鏈，ARM/AMD/INTC 是 CPU 與 host/control plane 故事，VRT/POWI/FLEX/BE 等則受惠於 AI data center power architecture。疑點是 CPU 是否真的能像 HBM 一樣形成「價」的超級週期；目前較確定的是「量」和敘事擴散，價格傳導仍需 2026H2-2027 的訂單與 ASP 證明。網路核查來源：NVIDIA Blog、MarketWatch、DCD、Counterpoint、Tom's Hardware。
   - 信心度：高

   - 來源：`on-x-today-20260422.md`；`on-substack-today-20260422.md`
   - 時效性：2026-04-21 至 2026-04-22 社群反覆出現；Google TPU 8t/8i 今日已由多家媒體確認。
   - 訊號：多則貼文指向 Google TPU 供應鏈從 Broadcom 主導走向 Marvell 參與，並稱 MRVL 因 Google 下一代推理晶片、光通訊 DSP、custom XPU / networking 敘事而被重新定位。另有貼文把 POET/SIVE 拉進 MRVL/CPO/laser upstream 的二階受益鏈。
   - 解讀 / 疑點：若市場相信 hyperscaler ASIC 會與 GPU 並行成長，MRVL/AVGO 的相對估值框架會改變，光互連、CPO、laser supplier 也可能被資金追逐。但 POET/SIVE 等社群訊號較多來自個人推文與片段訪談，缺乏公司正式訂單細節，應視為高波動弱確認，而不是已驗證基本面。對 NVDA 的解讀不是簡單 bearish，因為 NVIDIA 與 Google 同時擴大合作，Rubin/Blackwell 仍是 Google Cloud AI factory 的核心供給之一。
   - 信心度：中
