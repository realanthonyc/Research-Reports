### Long 看多

AMD
- Agentic AI 讓 CPU:GPU ratio 從 `1:4-1:8` 往 `1:1-1:2` 移動，CPU 不再只是 GPU 配角。
- AMD EPYC / chiplet / TSMC 供應鏈在外圍 CPU farms、RL 模擬農場與 synthetic-data workload 受益最直接。
- 觀察 Intel 4/23 財報是否確認 server CPU shortage 與 AMD share gain。

ARM
- Hyperscaler custom CPU、低功耗資料中心與 AI ASIC server CPU attach rate 是長期 royalty / licensing 敘事。
- 適合當 CPU shortage 的結構受益者觀察，不適合只用短線缺貨邏輯追。

Optical / CPO Basket
- `LITE`, `COHR`, `AAOI`, `GLW`, `MRVL`, `AVGO`, `APH`, `AMKR`, `TER`, `TSEM`
- Nvidia 對 Lumentum / Coherent / Marvell 的資本投入，讓 optical networking / CPO 從題材升級為 AI rack-scale 架構主線。
- 最乾淨邏輯：GPU / XPU / memory 之間的高頻寬低功耗互連成為新瓶頸；更好的研究方向是找 supply bottleneck、雷射、封測與量產良率。

AVGO
- Meta / Broadcom 官方合作確認 multi-year MTIA，初始承諾超過 `1GW`，向 multi-gigawatt rollout 延伸。
- 核心多頭不是單純 ASIC，而是 `XPU + Ethernet + SerDes + advanced packaging + CPO` 的系統級位置。

MRVL
- 如果 Google 導入第二供應商或新 AI chip / memory-processing-unit，MRVL 有 `Google optionality`。
- 同時具備 optical / CPO / custom silicon 敘事，彈性高於一般半導體配角。

Energy / Oil Hedge
- `XLE`, `USO`, `XOM`
- 不是單純看多油，是對 Hormuz -> oil -> CPI -> Fed repricing -> DXY / yields 壓力的 hedge。
- 若談判轉好或 Brent 跌回 `$90` 下方，追價風險高。

Gold / Treasury Volatility Hedge
- `GLD`, `GDX`, `NEM`, `TLT` watch
- Warsh hearing 若讓市場擔心 Fed independence / Treasury-Fed coordination，可能出現長端利率與 gold 同時上升的 policy credibility trade。

HIMS
- FDA peptide 變化是實際催化：12 種 peptide 從 Category 2 移除，市場開始交易 compounded peptide 合規路徑改善。
- 但仍不是全面放行；7 月 PCAC / 503A bulk list 才是下一個確認點。

### Short 看空

INTC relative to AMD
- Intel 短線可受惠 CPU 供給緊與 ASP，但長線仍受 18A 良率、產品週期、server share loss 牽制。
- 4/23 財報若只給短期 pricing tailwind、沒有清楚 18A / server roadmap，可能成為 semi breadth 的壓力點。

High-duration software / IGV failure
- IGV 反彈很強，但報告把它定性為可能的 short covering / relief，而非基本面修復。
- 若 IGV 無法收復關鍵區間、BTC 也不跟漲，軟體股反彈更像 late-cycle squeeze。

QQQ / high beta growth hedge
- 條件式看空：Brent `$100+`、DXY 上行、10Y 上破、VIX 擴張。
- 這種組合會把 oil shock 變成 Fed-path repricing，而不是短暫 headline。

Airlines / Consumer discretionary
- 高油價 + 消費韌性不足會同時打擊 margin 與 demand。
- 若 retail sales 強但油也強，利率壓 growth；若 retail sales 弱且油強，cyclicals / discretionary 會更差。

### Ambush 可埋伏

MRVL
- 若 Google / Marvell 供應商新聞被正式確認，MRVL 可能從 optical / custom silicon 第二梯隊變成主線。

LITE / COHR
- Nvidia 投資已確認，長線邏輯硬；若短線因擁擠交易回落，可找光源供應與毛利率確認後埋伏。

AAOI / GLW / APH
- 光纖、連接器、光模組周邊可能是第二波輪動，而不是第一天追最熱標的。

AMKR / TER
- 如果 CPO / optical / advanced packaging 往量產良率交易擴散，封測與測試設備可能補漲。

ARM
- 若市場把 CPU shortage 從 AMD / INTC 擴散到 Arm-based server CPU attach rate，ARM 可作為較長線 beta。

RKLB
- 只在 SpaceX IPO 高品質來源確認、或 Neutron execution 有新進展時加權；否則當 watchlist。

BTC vs IGV
- 若 BTC 突破並確認 IGV 反彈，risk-on 更可信。
- 若 IGV 彈、BTC 不動或下破，軟體反彈要當作流動性未確認訊號。
