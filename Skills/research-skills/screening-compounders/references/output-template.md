# Output Template

Use this order unless the user requests a different format.

Save the final report to:
- `Reports/research/screening-compounders-{YYYY-MM-DD}-00.md`

If the same date already exists, increment the suffix to `-01`, `-02`, and so on.

## 1. 研究範圍與方法

- sector / theme / industry definition
- geography and exchange scope
- analysis date
- fiscal years covered
- source hierarchy

## 2. 初始候選名單

For each company:
- ticker
- business description
- reason included in the initial universe

## 3. 步驟 1：拒絕清單

Create a company-by-company table:
- company
- pass / fail
- triggered veto item
- evidence
- source
- decision rationale

## 4. 步驟 2：ROCE 歷史篩選

For every surviving company:
- annual ROCE table
- average ROCE
- stability comments
- pass / fail

## 5. 步驟 3：財務極度穩健性檢查

Use a 5-check table and state how many passed.

## 6. 步驟 4：趨同進化商業模式驗證

For every surviving company:
- business model
- analogous winners
- why the model is externally validated
- pass / fail

## 7. 步驟 5：三個達爾文心法交叉驗證

Address:
- long-term value creation only
- historical facts only
- burdensome honesty signals

## 8. 步驟 6：綜合評分與排名

Include:
- score out of 10
- scoring breakdown
- rank
- summary judgment

## 9. 最終候選清單

For each final candidate include:
- ticker / company
- current price
- current valuation vs historical range
- moat description
- why it can remain great for a long time
- key reason to wait or size carefully

## 10. 主要風險與不確定性

- data gaps
- judgment calls
- any unresolved disputes in source material

## 11. 資料來源

- filings
- company materials
- market data sources
- reputable third-party corroboration links
