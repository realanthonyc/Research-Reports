## 高價值市場訊號

1. **AI 基礎設施瓶頸從 GPU 擴散到 CPU，AMD / ARM / INTC 的敘事被重新定價**
   - 來源：`CPU.md`、`The Great Rebalance How Agentic AI Is Reshaping the CPUGPU Ratio.md`、`下一個瓶頸是CPU？x86還是ARM？誰才是CPU概念股？ - 深入分析第43期：AMD，Intel，ARM.md`、`Arm-based CPUs to Capture 90% of AI ASIC Server CPU Market by 2029.pdf`
   - 時效性：資料多為 2026-04-21 收集，TrendForce 報告與新聞脈絡集中在 2026-03 至 2026-04；經網路檢查，TrendForce、IBD、Tom's Hardware 近期均有 CPU shortage / agentic AI CPU renaissance 相關報導。
   - 訊號：Agentic AI 的 tool use、orchestration、RL simulation 與 synthetic-data workflow 使 CPU:GPU ratio 從過去約 1:4 至 1:8，朝 1:1 至 1:2 甚至更高 CPU 密度移動。TrendForce 類資料指出 Intel / AMD server CPU 供給緊、交期拉長，市場開始把 CPU 視為 AI capex 的下一個結構瓶頸。
   - 解讀 / 疑點：這不是單純「CPU 漲價」新聞，而是 AI 系統架構的價值重分配。AMD 的 EPYC / chiplet / TSMC 供應鏈在外圍 CPU farms 與 RL 模擬農場有較乾淨的受益路徑；ARM 的長期紅利在 hyperscaler custom CPU、低功耗與授權滲透；Intel 短線可受惠供給緊與 ASP，但長線仍受 18A 良率、產品週期與 share loss 牽制。需要追蹤 Intel 4 月 23 日財報是否承認 CPU shortage 是訂單能見度而非一次性 pull-in。
   - 信心度：高

2. **光通訊 / CPO / silicon photonics 從題材交易升級為 AI rack-scale 架構主線**
   - 來源：`不只「相信光」，更要「看懂光」！高盛拆解光通訊產業鏈，這份全景圖值得收藏.md`、`光通信の波、まだ始まったばかり？！...md`、`Misc.md`、`on-x-today-20260421.md`
   - 時效性：收集時間為 2026-04-21；網路檢查確認 Reuters / TrendForce 報導 Nvidia 於 2026-03-02 投資 Lumentum、Coherent 各 20 億美元，並於 2026-03-31 與 Marvell 建立 20 億美元級策略合作。
   - 訊號：市場焦點從「誰有最多 GPU」轉為「誰能把 GPU / XPU / memory / server 以低功耗、低延遲、高頻寬連起來」。高盛相關摘錄把 optical networking TAM、CPO、OCS、SiPh、光源、連接器、封測與測試設備全鏈條拆開，受益名單反覆出現 `LITE`、`COHR`、`AAOI`、`GLW`、`MRVL`、`AVGO`、`APH`、`AMKR`、`TER`、`TSEM`。
   - 解讀 / 疑點：這條線與 CPU shortage 互相強化，指向 AI capex 的第二層受益者。短期交易已擁擠，尤其光模組與熱門美股標的可能容易 sell-the-news；但產業邏輯比單純 meme 更硬，因為 Nvidia 以資本投資鎖定光源與系統合作，Google / hyperscaler 也在公開架構中提高 optical switching / CPO 重要性。更好的研究方向是找 supply bottleneck 和量產良率，而不是只追已大漲標的。
   - 信心度：高

3. **Broadcom 的 Meta 官方合作已確認，但 Google / Marvell 傳聞讓 AVGO 多頭故事更需要分層**
   - 來源：`Broadcom再獲Google、Meta加碼合作：AVGO真的是下一個AI基建霸主？.md`、`Misc.md`
   - 時效性：Meta / Broadcom 官方新聞稿日期為 2026-04-14；Google / Marvell 討論為 2026-04-21 媒體報導，仍屬討論或傳聞層級。
   - 訊號：Broadcom 與 Meta 擴大多年期 MTIA 合作，官方確認初始承諾超過 1GW，並走向 multi-gigawatt rollout；Meta 與 Broadcom 均強調 Broadcom 參與 chip design、advanced packaging、Ethernet networking。另一方面，The Information / Times of India 相關報導稱 Google 可能與 Marvell 討論新 AI chip，這使「AVGO 完全壟斷 Google TPU」的敘事需要降一階處理。
   - 解讀 / 疑點：AVGO 的強項不只是 ASIC compute，而是 XPU + Ethernet + SerDes + packaging 的系統級地位；Meta 官方合作提高營收能見度。但若 Google 開始引入 MRVL 或多供應商策略，AVGO 仍可受益於整體 custom silicon TAM，卻不能用單一客戶長約無風險外推。對交易而言，AVGO 多頭仍成立，但 MRVL 作為「第二供應商 / optical-CPO / Google optionality」的彈性可能被市場重新定價。
   - 信心度：中高
