# 報告呈現模板 (Report Presentation Template)

複製以下模板作為最終輸出骨架，確保專業性與可讀性。

## 詳細總結

- 用 2-4 段交代：
  - 剛剛或最近最重要的 happenings 是什麼
  - 市場接下來在等什麼
  - 這些資訊如何傳導到基本面、預期差與估值
  - 你的明確判斷與預測

## 總結概要

- 用 4-8 點快速講清楚：
  - 主要投資判斷
  - 最重要支持邏輯
  - 最重要風險
  - 接下來最值得追蹤的事件或數據

## 投資論點與策略建議（加強版）

### Bull / Base / Bear 論點
- Bull：至少 3 個可驗證成立條件 + 1 個失效條件
- Base：最可能情境與其估值中樞邏輯
- Bear：至少 3 個下修觸發點 + 風險傳導路徑

### Swing（數天到 1 個月）
- 交易偏向：`偏多 / 中性 / 偏空`
- 進場條件（至少 2 項）
- 減碼或退出條件（至少 2 項）
- 風險控管原則（倉位與失效條件）

### 中短期（1-6 個月）
- 配置偏向：`加碼 / 觀望 / 減碼`
- 轉多必要條件（至少 3 項）
- 轉空必要條件（至少 3 項）
- 組合角色定位（核心持股/衛星部位/事件驅動）
