## 7. 投資論點與策略建議（加強版）

### 7.1 Bull / Base / Bear 論點
- Bull：至少 3 個可驗證成立條件 + 1 個失效條件
- Base：最可能情境與其估值中樞邏輯
- Bear：至少 3 個下修觸發點 + 風險傳導路徑

### 7.2 Swing（數天到 1 個月）
- 交易偏向：`偏多 / 中性 / 偏空`
- 進場條件（至少 2 項）
- 減碼或退出條件（至少 2 項）
- 風險控管原則（倉位與失效條件）

### 7.3 中短期（1-6 個月）
- 配置偏向：`加碼 / 觀望 / 減碼`
- 轉多必要條件（至少 3 項）
- 轉空必要條件（至少 3 項）
- 組合角色定位（核心持股/衛星部位/事件驅動）
