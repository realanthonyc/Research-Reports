## 觀察名單 / 弱訊號

- **AAPL / GOOGL / MSFT / AMZN**：高信心 earnings quality basket；進場條件是 2Y yield / DXY 不同升，否則只能短線看強不追高。
- **SMH / NVDA / AVGO / AMD / QCOM**：AI infrastructure 仍強，但短線需要 ISM 與 DXY 確認。QCOM / custom silicon / inference CPU 題材值得跟，但需財報細節驗證。
- **WDC / SNDK / MU / STX**：memory、enterprise SSD、AI inference storage intensity 是較有二階 alpha 的方向；若財報後 sell-the-news 但毛利與長約仍強，回踩 RR 較好。
- **PWR / CAT / MPWR / GLW / COHU / AXTI / BESI**：AI power、industrial equipment、optics、thermal test、hybrid bonding 是高潛力延伸鏈；小票與海外標的流動性、估值與資料可靠度需另查。
- **XLE / OIH / GLD / GDX**：oil / Fed / policy-error hedge；適合在 ISM prices paid 偏熱、油價再跳、DXY / 2Y 同升時提高權重。
- **IWM / COIN / MSTR / unprofitable growth**：在 DXY / 2Y 偏強時仍是較脆弱區域；只有在 DXY / 2Y 同跌、VIX 壓縮、IWM 跟上 QQQ 時才轉為 squeeze 候選。
- **GS / BX / private markets**：Substack 提到 Goldman / 私募判斷與 Blackstone earnings，但資訊量不足；可作 private credit / alternatives sentiment 的後續研究入口。
